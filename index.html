GitHub Network

This is a demo of my custom clone of the GitHub network explorer. It is based on pure canvas without any external dependencies (jQuery is loaded on this page but is not used by the graph).

The library is composed of three components :

  • Network renders the full graph (that's the image you can download bellow)
  • NetworkView displays a small part of the network with axis, drag-and-drop, commit messages, etc.
  • GithubNetwork loads data from GitHub (with the help of JSON2JSONP) and instanciate a view

Only repositories with less than 300 forks are displayed. The library is missing a lot of error check for now.
